# AgroResiliência BR - Agrinho Programação 2026

## 🌾 Objetivo do Projeto
Este projeto foi desenvolvido especialmente para o Concurso Agrinho 2026. O objetivo principal do site é conscientizar e educar produtores rurais e o público geral sobre a vulnerabilidade da agricultura brasileira frente às mudanças climáticas. O site apresenta os riscos regionalizados do país e propõe soluções práticas e sustentáveis alinhadas ao Plano ABC+ do Governo Federal, contando com ferramentas interativas de acessibilidade e um simulador de resiliência agrícola.

## 🚀 Tecnologias Utilizadas
O projeto foi construído do zero, utilizando tecnologias web nativas para garantir o cumprimento das regras do concurso (sem o uso de frameworks ou bibliotecas externas):
- **HTML5:** Estruturação semântica do conteúdo.
- **CSS3:** Estilização moderna, uso de CSS Grid, Flexbox, variáveis nativas para controle de temas e animações de interface.
- **JavaScript (ES6+):** Manipulação dinâmica do DOM, sistema de abas interativas, lógica do simulador de propriedade e recursos de acessibilidade.

## 🛠️ Funcionalidades e Instruções de Uso
1. **Navegação Interativa por Abas:** Na seção 2, clique nos botões das regiões (Centro-Oeste, Sudeste, Nordeste, Sul) para mapear dinamicamente os riscos e consequências climáticas de cada local.
2. **Cards de Estratégias Tecnológicas:** Apresentação visual e explicativa de técnicas como ILPF, Plantio Direto, Agroflorestas e Melhoramento Genético.
3. **Simulador de Resiliência:** Insira a sua região e a quantidade de tecnologias sustentáveis adotadas na propriedade fictícia para calcular o nível de adaptação climática (Baixo, Médio ou Alto).
4. **Painel de Acessibilidade:**
   - Use os botões **A+** e **A-** no menu superior para aumentar ou diminuir o tamanho dos textos em tempo real.
   - Clique no ícone da lua/sol para alternar instantaneamente entre o **Modo Claro** e o **Modo Escuro**, que salva a preferência do usuário localmente no navegador.

## 🎨 Ferramentas de Design e Customização de Mídias
Para evitar layouts genéricos e garantir mídias adequadas ao tema, as imagens utilizadas nas estratégias tecnológicas e o banner principal passaram por tratamento, corte e otimização utilizando a ferramenta **Canva** e o editor **Paint**, garantindo uma identidade visual coesa e autoral para o projeto.
